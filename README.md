# Nexo-Web

Company portfolio and brand site for a web design agency.

## Tech Stack
- HTML5
- CSS3 (embedded)
- JavaScript (inline)
- Google Fonts (Syne, Instrument Sans)

## Setup
Open index.html in a web browser to view the site.

## Structure
- index.html: Main portfolio page
- clientes/: Placeholder for client work
- plantilla/: Placeholder for templates
